Buzzsaw Cat

A 2D action game made in Unity.

This project was for Game Mechanics. The goal was to recreate a top-rated Ludum Dare Compo entry. I picked Buzzsaw Cat, because it was simple with interesting mechanics and I figured if I picked a simple game, I could always polish it more at the end. I was tasked with mimicking the art style, music and mechanics that the original game had. Aside from adding additional visual polish and feedback, I enhanced the dash mechanic by programming a segmented dash trail that fades over time. I also learned to work with DOTween library, which is a great tool for procedural animations.
